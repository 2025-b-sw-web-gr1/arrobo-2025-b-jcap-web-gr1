CSS · Carga de estilos

Formas de cargar estilos en una página web

Ejemplos educativos de las distintas técnicas: atributo style, etiqueta <style>, hoja externa, inyección con JavaScript y carga por petición HTTP.

1. Atributo style (inline)

Aplica estilos directamente al elemento. Útil para casos puntuales, pero no recomendado para grandes sitios.

Caja con estilo inline
<div style="background:#10b981; color:white; padding:12px; border-radius:8px;">...</div>

2. Etiqueta <style> en el documento

Define reglas CSS dentro del HTML. Útil para demos o páginas pequeñas.

Caja estilizada con <style> interno
<style>
.demo-style-tag {
  background: #3b82f6;
  color: white;
  padding: 12px;
  border-radius: 8px;
}
</style>

3. Hoja externa (archivo CSS)

Separa presentación del contenido. Escalable y recomendado para proyectos reales.

Caja estilizada por style.css
<link rel="stylesheet" href="style.css">

4. Inyección con JavaScript

Genera reglas al vuelo creando una etiqueta <style> o adjuntando una hoja externa.

Caja a estilizar dinámicamente
// app.js
injectStyle(`
  #inject-target { background:#f59e0b; color:black; padding:12px; border-radius:8px; }
`);

5. Cargar por petición HTTP (fetch)

Descarga CSS desde el servidor y lo aplica en tiempo de ejecución.

Caja a estilizar vía fetch
// app.js
fetch('/dynamic.css').then(r => r.text()).then(css => injectStyle(css));

Servidor web local

Usaremos http-server para servir estos archivos y poder probar la carga por HTTP.

  1. Opción rápida sin instalar: npx http-server . -p 8080
  2. Opción con scripts (recomendada):
    npm install en 02-CSS y luego npm run start
  3. Abre http://localhost:8080 en tu navegador.
